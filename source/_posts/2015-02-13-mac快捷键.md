mac的键盘布局以及常用快捷键

<!--more-->

官方很全面的[快捷键说明](http://support.apple.com/zh-cn/HT201236)，这里节选了一些常用的信息。

* 符号代表的意思 

```
⌘	Command key 
⌃	Control key
⌥	Option key 也就是alt键
⇧	Shift Key
⇪	Caps Lock
fn	Function Key
```
* 复制，剪切，粘贴

 ```
 Command-C
 Command-X
 Command-V
 ```
* 常用技能节选

 ```
Command+Q                             关闭当前程序
Command+W                             关闭当前窗口
Command+M                             窗口最小化,之后不能Command+Tab切出来，隐藏音乐播放器之类的
Command+H                             隐藏窗口
Command+N                             新建窗口
Command+Shift+4                   	  截图(自由选取范围)
Command+1                             以图标方式显示
Command+2                             以分栏方式显示
Command+3                             以列表方式显示
Command+4                             以Cover Flow方式显示
enter                                 重命名文件、文件夹
 ```
